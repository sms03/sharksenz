## 🔍 About SharkSenz

**SharkSenz** is your ultimate business toolkit - inspired by the analytical precision of Shark Tank investors. Transform your business insights with AI-driven analytics, comprehensive market research, and essential metrics calculation, then perfect your pitch with our interactive simulator.

> *"In the tank of business, only those with the right data survive. SharkSenz gives you the edge to not just swim, but dominate."*

## ✨ Features

### 🧮 Business Metrics Dashboard
Calculate and visualize critical business metrics including:
- Revenue and profit projections
- Customer acquisition cost (CAC)
- Burn rate and runway analysis 
- Business valuation models
- Growth trajectory visualization

### 🤖 AI-Driven Insights
- Get personalized recommendations based on your metrics
- Identify growth opportunities and potential pitfalls
- Compare your metrics against industry standards
- Predictive analytics for future business performance

### 📊 Market Research Center
- Comprehensive industry analyses
- Competitor benchmarking
- Market size and growth projections
- Target audience insights and segmentation

### 🎤 Interactive Pitch Simulator
- Step-by-step pitch builder
- Real-time feedback on your pitch elements
- Simulated investor questions
- Performance evaluation and scoring

### 📚 Business Learning Hub
- Comprehensive business education modules
- Interactive lessons on key business concepts
- Knowledge assessment and certification
- Growth tracking with XP and achievement system

## 🛠️ Tech Stack

- **Frontend Framework**: React with TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS with shadcn/ui components
- **Animations**: Framer Motion, GSAP
- **Data Visualization**: Recharts
- **Backend Services**: Supabase
- **State Management**: React Query
- **Form Handling**: React Hook Form with Zod validation

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or bun

### Installation

```bash
# Clone the SharkSenz repository
git clone https://github.com/yourusername/SharkSenz.git

# Navigate to the project directory
cd SharkSenz

# Install dependencies
npm install
# or with bun
bun install

# Start the development server
npm run dev
# or with bun
bun run dev
```

Visit `http://localhost:5173` to see the application running.
